# tt-dashboard
> Basic SPA Dashboard with Node (bck) and VUE (frnt)

![alt tag](https://github.com/TalentTools/tt-dashboard/blob/master/tt-dashboard-preview.png)

## / BACKend
``` bash
# install dependencies
npm install

# run server
nodemon server.js
```

## / FRONTend

``` bash
# install dependencies
npm install

# run dev
npm run dev

http://localhost:8080/
```
